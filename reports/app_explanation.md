# 📱 Application Explanation (Streamlit + Power BI)

## 🧠 Overview
An interactive analytics application designed to provide insights into retail business performance.

---

## ⚙️ Workflow
1. Load dataset
2. Preprocess data
3. Apply ML model (demand prediction)
4. Display insights via dashboard

---

## 📊 Features

### 1. Executive Dashboard
- KPIs overview
- Revenue trends
- Category distribution

### 2. Sales Analytics
- Monthly sales trend
- Regional analysis
- Product performance

### 3. Inventory Analytics
- Stock levels
- Risk classification
- Turnover analysis

### 4. Pricing Analytics
- Price vs demand
- Competitor comparison
- Discount impact

### 5. Demand Forecasting
- Forecast vs actual comparison
- Error distribution
- Seasonal trends

---

## 🤖 Prediction Logic
- Input features:
  - Price
  - Inventory Level
  - Seasonality
  - Region
- Output:
  - Demand Forecast
  - Inventory Risk

---

## 🎨 UI Explanation
- Clean multi-page layout
- Sidebar navigation
- Interactive filters:
  - Date
  - Region
  - Category
- Real-time visual updates

---

## 🚀 Deployment
- Built using Streamlit
- Can be deployed on:
  - Streamlit Cloud
  - Render
  - AWS
